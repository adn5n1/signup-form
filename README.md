# Interactive Signup Form (JavaScript, HTML/CSS)

A simple web-based signup form built using HTML, CSS, and JavaScript.  
The project focuses on JavaScript fundamentals such as event handling, input validation, and DOM manipulation.

## Features
- Username, email, and password validation
- Client-side error handling
- Password rules (length, uppercase, number)
- Real-time feedback messages

## Skills Demonstrated
- JavaScript
- HTML/CSS
- DOM manipulation
- Event handling
- Conditional logic
- Input validation
